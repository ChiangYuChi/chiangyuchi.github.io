常用終端機指令

在 Github 上開新專案

上傳檔案到 Github 前，先在 Github 開一個新專案

接著設定專案名稱，Repository name 可填寫任意名稱，只要不重複即可。存取權限選擇 Public 可免費使用。

按下 Create repository 後會進入這個畫面

  1. 若是建立新專案，則按照「create a new repository on the command line」的內容指示進行；如果是要上傳現存專案，則依照「push an existing repository from the command line」選項進行。
  2. 畫面中間上面，有兩個按鈕可以切換，分別是「HTTPS」以及「SSH」。

Git 基礎指令

  1. 進入要上傳的資料夾內，開啟終端機，首先，先建立一個 README.md 檔案：
    $echo “# git-practice” >> README.md

README.md 是 GitHub 專案的預設說明頁面，附檔名 .md 表示它是一個 Markdown 格式

  1. 初始化 git 專案
    先把路徑指到該專案資料夾 cd your_project_name 初始化
    $git init
  2. git add 要新增的檔案
    $git add README.md
    如果要新增複數檔案輸入 git add .
  3. 為這次 commit 簡短備註
    $git commit -m “first commit”
  4. 建立遠端 Git 伺服器節點。
    首先，需要設定一個端節的節點，讓他變成 default，例如：
    $git remote add origin https://github.com/ChiangYuChi/git-practice.git
  5. git push 建立分支
    把 master 這個分支的內容，推向 origin 這個位置
    $git push -u origin master
  6. $git log
    確認剛剛的行為有沒有成功
  7. 切換 branch
    $git checkout “branch name”

使用 Github 展示靜態網頁

  1. 建立 branch
    名稱需為 gh-pages，指令如下
    ＄ git branch gh-pages
  2. 切換到 gh-pages 分支
    $ git checkout “gh-pages”
  3. push 上 github
    $ git push -u origin gh-pages
    之後只要 git push 即可
  4. 到 github_name.github.io/repository_name/查看，即可看到網頁
    首頁必須要命名為 index.html
