JavaScript 練習 - 計算機

2019-02-10 PV:

Demo

JavaScript 是一門基於原型、函式先行的程式語言，支援物件導向。簡言之，是一種可在網站裡加入動態效果的程式語言，例如更改樣式、展示動畫等

Introduction 介紹

基本 HTML 結構

基本的 HTML 結構如下，可以看到輸入欄位、運算符號及數字鍵，其中運算符號上面具備 data-action 屬性以便點擊時取得相應功能。

基礎功能

使用事件委派監聽

監聽所有點擊事件，在此使用事件委派監聽，因為所有按鍵都是 keys 的子層，並且使用 matches() 確定按下的鍵的類型。

利用 data-action 與判斷式，決定點擊目標的類型。一個簡單的計算機可能發生以下幾種狀況：

  • 數字鍵 （0-9)
  • 運算符 an operator key (+, -, ×, ÷)
  • 小數點
  • 等於
  • 清除

取得輸入值並顯示在計算機

計算機都會有一個開機預設的 0，如果按下任何一個數字鍵，就會取代目前銀幕上的 0。如果銀幕上現在是非 0 的其他數字，按下第二個數字鍵，就會附加在目前顯示的數字尾巴。所以這裡需要獲得 2 項資訊：

  • 目前點擊數字
  • 目前銀幕顯示數字

透過 textContent 屬性取得上面兩項資訊的值

是 0，取代現在顯示的數字；不是 0 ，附加在後面

點擊小數點

點擊小數點的時候，小數點會附加在當下計算機顯示的數字後面

按下運算符

按下運算符的時候，按鈕會顯示高光顯示運算符現在運作中。透過新增 class 屬性達到效果

按下運算符後再輸入數字

  • 第二個數字取代目前銀幕顯示數字
  • 運算符的高光狀態解除

使用 Array.from() 建立一個類陣列，並且如果其中有任何一個 is-depressed css 屬性就移除

在顯示第二個數字之前，必須知道是不是已按下運算符。所以在點擊運算符的時候，也新增一個 data-previous-key-type="operator" 元素屬性，以供判斷是否按下運算符；

如果按下第二個數字前，曾按下運算符，取代目前銀幕顯示數字

按下等號

按下等號並顯示計算結果之前，時需先取得三種值：

  • 第一個輸入數字
  • 運算符
  • 第二個輸入數字

在輸入運算符之前，第一個輸入數字已被儲存，運算符的種類也用同樣方式儲存

取得這三個值就可以開始進行運算

建立運算函式，傳入第 1 個數字、運算符、第二個數字。目前都還是字串形式，此處必須先把字串轉成帶小數點的數字

parseInt :將字串轉為整數
parseFloat : 字串轉為帶小數點的數字
