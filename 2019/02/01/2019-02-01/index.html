JavaScript 基礎（一）- 變數與資料型別

2019-02-01 PV:

變數

變數儲存一些工作執行時所需要的資料，變數就像一個裝東西的容器，變數的名稱是貼在容器上面的標籤，瓶子裡面裝的就是變數的值。變數本身不具型態類別，但它裡面的值有不同的型別 。程式運行的時候，儲存在變數裡面的資料是可以變更的。

var (variable) 用在宣告一個變數的定義，令名稱指向某個值 。

資料型別

型別 (Data Type) 就像容器裡面裝的不同東西

自動型轉

強等於與弱等於

兩個等號 == 的比較運算式下，若是雙方的資料類型不同，會進行「自動型轉」。三個等號與兩個等號雖然都是比較的意思，三個等號 === 不會替數值自動型轉，而 == 會自動型轉。不等於也有 != 與！== 兩種。！= 會型轉，！== 不會。

字串與數字的型轉

如果遇到字串與數字比較，會用 Number () 把字串轉為數字再進行比較

布林的型轉

如果有一個值是「Boolean」，會將 true 轉為數字的 1，false 轉成 0；如果字串和布林相加，會把字串轉為數值或維持字串。如果字串和布林值相減則結果是

null 與 undefined

number、boolean、object 型轉的時候會呼叫 toString()去取得對應的字串。
。但是 null 與 undefined 則透過 String() 分別轉為 null 與 undefined。

NaN 字面上意思是 Not a Number，但如果用 typeof 運算子判斷型態，會是 number

NaN 與任何數字運算結果都會是 NaN。NaN 不等於任何數字，甚至是自己。

如何判斷變數是不是 NaN，可透過 isNaN(value)判斷
