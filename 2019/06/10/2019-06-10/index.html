常見的陣列運用方式

2019-06-10 PV:

Array 是 Javascript 很重要的環節，array 物件有以各種方式操作陣列的方法，如合併、反轉、排序，都是常見的實用操作。因此從 JavaScript 30 Days 這個課程裡面整理了一些常見的實用操作：

範例 1

Array.prototype.filter()

過濾的方法，就是傳入一個函數，並且以迴圈檢查運算每個物件，由原陣列中通過該函式檢驗之元素所構成的新陣列

Array.prototype.map()

map() 方法會建立一個新的陣列，其內容為原陣列的每一個元素經由回呼函式運算後所回傳的結果之集合。

Array.prototype.sort()

sort() 方法會原地對一個陣列的所有元素進行排序，並回傳此陣列。

Array.prototype.reduce()

reduce() 方法將一個累加器及陣列中每項元素（由左至右）傳入回呼函式，將陣列化為單一值。
