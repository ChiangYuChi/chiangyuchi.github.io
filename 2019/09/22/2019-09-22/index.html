1. Introduction 介紹

這是 JavaScript 操作 DOM，並搭配新增 class 切換玩家，配合隨機產生的數字，實現擲骰子遊戲。

  • 設定變數
  • 需兩方輪流操作，按照隨機出現的數字獲取分數，並加入總分
  • 判定兩方誰先達到 100
  • 須判定擲出 1 ，則該輪不算數
