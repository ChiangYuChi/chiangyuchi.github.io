C# - 建構子

2020-09-30 PV:

問題:

  1. 建構物件要寫多行程式碼
  2. 重複設定同樣的數值

Constructor

  • Method 的一種
  • 物件被建立的時候執行
  • 沒有 output
  • 名稱與 class 名稱相同
  • 如果知道某個屬性的初始值為多少，可寫在建構子內，讓物件預先設定好
  • 建構子可以給予 input(引數、輸入值)用來幫助設定物件
  • 建構子的 input 要在建立物件時給予
  • Overloaded 多載
    不同情況需要不同建構子。可以使用物件導向的其中一個特徵「多載」。

可寫兩個名稱一樣的建構子，但是接受不同數量的 input

  • 有建構子就不能使用系統預設建構子
