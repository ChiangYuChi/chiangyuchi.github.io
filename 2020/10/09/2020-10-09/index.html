C# - Get and Set 存取器

2020-10-09 PV:

語法

……… // 宣告變數(通常是 public)
{ get { …. } // 希望變數讀取時執行的程式碼
set { …. } // 希望數值存入時執行的程式碼
}

如果只留下 get 存取器，該變數就會變成「唯讀」

Get & Set 存取器可以用來即時運算一些平常大家認為是變數的數值，像是 Money
